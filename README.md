# Driver Assistance

# 1. Công nghệ sử dụng

* Android

* Xamarin C#

* Google Map API

* OpenCV Android

* Tìm kiếm đường đi giữa 2 địa điểm.

* Phối cảnh làn đường.

* Phát hiện biển báo giao thông.
